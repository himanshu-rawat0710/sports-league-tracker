# AI Usage Disclosure

This document outlines the use of AI tools in the development of the Sports League Tracker project.

### Tools Used
* **LLM (Gemini):** Utilized as a technical partner for architectural planning, optimizing React Query hydration patterns, and troubleshooting CSS Grid behaviors.

### Implementation Details
* **Architecture:** AI assisted in structuring the "Hydration Boundary" pattern to ensure a seamless handoff between Next.js Server Components and the client-side cache.
* **Component Design:** Leveraged AI to quickly scaffold the atomic Design System (Button, Input, Select) to ensure consistent UI across the application while maintaining clean, reusable code.

### Verification
All code generated or suggested by AI was manually reviewed, refactored for type safety, and tested to ensure it meets the specific requirements of the assignment.
